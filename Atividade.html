3) QUAL CRIAR UMA ABREVIAÇÃO OU UM ACRÔNIMO?

O Elemento _HTML <-abbr-> _(ou Elemento de Abreviação HTML) representa uma abreviação e opcionalmente fornece uma descrição completa para ela. Se presente, o atributo title deve conter a descrição completa e apenas ela.

4) QUAL É A MANEIRA CORRETA DE ESCREVER O ENDEREÇO EM UM DOCUMENTO HTML?

A tag <-address-> define as informações de contato do autor / proprietário de um documento ou um artigo. Se o elemento <-address-> está dentro do elemento , representa informações de contato para o documento.

10) COMO ESPECIFICAR A URL/ALVO BASE PARA TODAS AS URLs RELATIVAS EM UM DOCUMENTO?

Uma URL relativa localiza um recurso usando uma URL absoluta como ponto de partida. Na verdade, a "URL completa" do destino é especificada pela concatenação das URLs absolutas e relativas. Normalmente, uma URL relativa consiste apenas no caminho e, opcionalmente, no recurso, mas em nenhum esquema nem servidor.
